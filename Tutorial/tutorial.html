The GAVisTool Tutorial

Getting Started

To get started, you need the GAVisTool software installed on your computer. This is as simple as copying down the executable from here. At the time of this writing, only windows is supported. If you prefer to download the source code and compile the project, you can do that by visiting the GAVisTool project on git-hub here, and then simply clone the repository. Alternatively, you can download a ZIP file of the project from here, or a tar-ball from here.
